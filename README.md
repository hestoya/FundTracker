# 💰 FundTracker

FundTracker is a modern web application that helps you manage your personal finances in one place. It allows you to track savings across multiple bank accounts, record transactions, and set financial goals to stay on top of your money.

---

## 🚀 Features

- 🏦 **Multi-Account Tracking**  
  Monitor balances from different bank accounts in a unified dashboard.

- 📊 **Transaction Management**  
  Record and organize income and expenses easily.

- 🎯 **Savings Goals**  
  Set financial targets and track your progress over time.

- 📈 **Progress Monitoring**  
  Visualize your savings growth and spending habits.

- 💻 **Responsive Design**  
  Optimized for both desktop and mobile devices.

---

## 🛠️ Tech Stack

- **Blazor** – Interactive frontend using .NET  
- **Bootstrap** – Responsive UI design  
- **HTML5** – Structure  
- **CSS3** – Styling
